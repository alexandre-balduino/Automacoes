- Detector de arquivos duplicados
- Renomeador inteligente
- Lixeira simulada

### 🟡 Nível 2 — JSON e Dados
- Cadastro de usuários
- Sistema de logs

### 🔵 Nível 3 — Automação avançada
- Organizador de downloads
- Leitor de CSV

## 🚀 Objetivo
Evoluir habilidades de automação até nível profissional.
