# 🗑️ Simulador de Lixeira Avançada

## 🧠 Objetivo
Criar um sistema em Python que simula o funcionamento de uma lixeira, permitindo mover arquivos, restaurá-los e excluí-los permanentemente, oferecendo mais controle sobre operações de exclusão.

---

## ⚙️ Funcionalidades
- [x] Enviar arquivos para a lixeira (mover arquivos)
- [x] Criar automaticamente a pasta `lixeira/`
- [x] Listar arquivos presentes na lixeira
- [x] Restaurar arquivos
- [x] Deletar arquivos permanentemente
- [ ] Salvar caminho original dos arquivos
- [ ] Confirmar ações críticas
- [ ] Evitar conflitos de nomes

---

## 🧪 Tecnologias utilizadas
- Python 3
- Biblioteca `os`
- Biblioteca `shutil`
